# GuidEM — Student Career Guidance Platform

GuidEM is a mobile application built with **React Native (Expo)** and **Supabase** designed to help high school students in Israel discover their ideal career path. It offers a structured personality questionnaire, personalized career recommendations, and an AI-generated learning roadmap (Duolingo-style path) composed of real-world external courses, articles, tutorials, and videos.

---

## 🚀 Features

### 1. RIASEC Personality Questionnaire
* A 48-question Holland Codes (RIASEC) interest quiz.
* Computes primary and secondary personality types (e.g., Realistic, Investigative, Artistic).
* Dynamically recommends career goals based on the Holland Code configuration.
* Displays a compact "Completed" results view once taken, with the option to retake.

### 2. Guided Learning Roadmaps (Duolingo-Style)
* Generates learning plans **one unit at a time** to build the student's toolset.
* Each unit contains 5–10 steps (lessons, tasks, resources, and reflections).
* Steps contain **direct external links** to real-world resources (e.g., Google Certification courses, Chrome DevTools docs, YouTube videos, LinkedIn search targets) that open in the native browser or external apps.
* Completing a unit unlocks branching career specialization choices generated by the AI to further narrow the career path.

### 3. Live AI & Mock Generators
* **Mock Mode**: Deterministic, offline-compatible path generation with realistic web link payloads for easy testing.
* **LLM Mode**: Integrates a Deno-based Supabase Edge Function that leverages the Google Gemini API to write highly personalized learning steps matching the student's city, grade, school majors, personality type, and prior choices.

### 4. Developer Tools
* Accessible from the Profile tab under **Developer Tools**.
* "Reset Learning Path" option: Clears active units and progress summaries, resetting focus to the starting point of the career goal while preserving profile settings and questionnaire results.

---

## 🛠️ Tech Stack
* **Frontend**: React Native, Expo (SDK 51+), Expo Router, TypeScript, StyleSheet.
* **Backend**: Supabase (Authentication, Postgres Database, Row-Level Security, Edge Functions).
* **AI Engine**: Google Gemini API via Supabase Edge Functions.

## 💻 Setup and Installation

### 1. Clone the Repository
```bash
git clone https://github.com/galorariel/GuidEM.git
cd Student-Guidance_app
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Setup Environment Variables
Create a `.env` file in the project root:
```env
EXPO_PUBLIC_SUPABASE_URL=https://your-project-ref.supabase.co
EXPO_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
EXPO_PUBLIC_USE_LLM_GUIDE=false
```
*Set `EXPO_PUBLIC_USE_LLM_GUIDE=true` when you are ready to switch from local mock roadmaps to live AI generation.*

---

## 🗄️ Supabase Database & Backend Setup

To run the application, your Supabase project must be initialized with the database schemas:

### 1. Database Migrations & Seeds
Run the migration SQL scripts located in `supabase/migrations/` sequentially in your Supabase SQL Editor:
1. `20260712000000_init.sql` (auth profiles and saved items)
2. `20260713000000_catalog.sql` (career and activity catalog)
3. `20260714000000_profile_fields.sql`
4. `20260714010000_career_goal.sql`
5. `20260715000000_careers_holland.sql` (Holland code column for careers)
6. `20260716000000_guide.sql` (roadmap structures)
7. `20260718000000_enhanced_guide.sql` (career specialization tracking)

Run the SQL seed scripts to populate the database:
- Execute contents of `supabase/seed_catalog.sql` to import careers.
- Execute contents of `supabase/seed.sql` to import sample extracurricular activities.

## 🤖 Deploying the AI Edge Function (Optional)

If you wish to enable live AI roadmaps (`EXPO_PUBLIC_USE_LLM_GUIDE=true`), deploy the Edge Function:

### 1. Login to Supabase CLI
```bash
npx supabase login
```

### 2. Link your Project
```bash
npx supabase link --project-ref your-project-ref
```

### 3. Deploy the Function
```bash
npx supabase functions deploy generate-unit --no-verify-jwt
```

### 4. Configure Gemini API Key
Obtain an API key from Google AI Studio. Add it to your Supabase project secrets:
```bash
npx supabase secrets set GEMINI_API_KEY=your_google_gemini_api_key
```
Alternatively, set the variable in the **Supabase Dashboard** under *Project Settings -> Edge Functions -> Environment Variables*.
